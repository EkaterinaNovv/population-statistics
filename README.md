Проект "Статистика населения России"

Команда
- Участник 1 — загрузка и обработка данных (population_data.py)
- Участник 2 — интерфейс программы (gui_app.py)
- Участник 3 — построение графиков и прогноз (plot_forecast.py)

Описание
Программа позволяет:
- Загрузить CSV-файл с данными о численности населения России за последние 15 лет  
- Отобразить таблицу с данными  
- Построить графики зависимости от года  
- Рассчитать максимальный прирост и убыль  
- Выполнить прогноз на N лет методом скользящей средней  

Используемые технологии
Python, Tkinter, pandas, matplotlib.
